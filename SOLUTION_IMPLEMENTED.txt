HOW IT WORKS
================================================================================

Automatic Failover Strategy:
1. Try yfinance (yf.download)
2. Try yfinance (Ticker.history)
3. Try yfinance (extended range)
4. Try Alpha Vantage API (FREE)
5. Return error

Benefits:
✓ yfinance tried first (no rate limit)
✓ Alpha Vantage only used as fallback
✓ Hybrid approach maximizes reliability
✓ No more 429 errors

================================================================================
TROUBLESHOOTING
================================================================================

"429 error responses"
  - This is normal - yfinance is rate-limited
  - Alpha Vantage fallback will handle it
  - No action needed

"Alpha Vantage rate limit exceeded"
  - Free tier: 5 calls/min, 500 calls/day
  - Wait a few minutes
  - Or get paid API key

"No data found"
  - Check symbol is valid (MSFT not msft)
  - Check date range is valid
  - Try a different symbol

"API key invalid"
  - Verify key from alphavantage.co
  - Check environment variable is set
  - Restart backend
