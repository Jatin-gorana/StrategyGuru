METHOD 2: MANUAL SETUP (5 MINUTES)
================================================================================

Step 1 - Backend Setup:
  python -m venv venv
  source venv/bin/activate  # macOS/Linux
  venv\Scripts\activate.bat # Windows
  pip install -r requirements.txt

Step 2 - Frontend Setup:
  cd frontend
  npm install
  echo "NEXT_PUBLIC_API_URL=http://localhost:8000" > .env.local
  cd ..

Step 3 - Start Backend (Terminal 1):
  source venv/bin/activate  # macOS/Linux
  venv\Scripts\activate.bat # Windows
  python backend/main.py

Step 4 - Start Frontend (Terminal 2):
  cd frontend
  npm run dev

Step 5 - Open Browser:
  http://localhost:3000

================================================================================
TROUBLESHOOTING
================================================================================

Backend won't start:
  lsof -i :8000  # macOS/Linux - check if port in use
  netstat -ano | findstr :8000  # Windows
  python backend/main.py --port 8001  # Use different port

Frontend won't start:
  cd frontend
  rm -rf node_modules package-lock.json
  npm install
  npm run dev

API connection error:
  curl http://localhost:8000/health  # Verify backend running
  cat frontend/.env.local  # Check API URL
