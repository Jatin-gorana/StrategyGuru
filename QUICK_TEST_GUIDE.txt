================================================================================
TROUBLESHOOTING
================================================================================

"Fetching daily data (attempt 1/3)"
  → Normal, system is fetching data

"Rate limited, waiting 5 seconds..."
  → Normal, system will retry automatically

"No daily time series data found"
  → Check symbol is valid (RELIANCE.BSE not reliance.bse)
  → Check date range is valid
  → Try a different symbol

"API key invalid"
  → Verify key in backend/.env
  → Check for typos
  → Restart backend

"Timeout"
  → Check internet connection
  → Try again
  → Check Alpha Vantage status

"No data in date range"
  → Check date range is valid
  → Try a different date range
  → Check if stock existed during that period
