================================================================================
ENDPOINT DETAILS
================================================================================

URL:
  https://www.alphavantage.co/query

Parameters:
  function=TIME_SERIES_DAILY
  symbol=RELIANCE.BSE (or AAPL, GOOGL, etc.)
  outputsize=full
  apikey=YOUR_KEY

Response Format:
  {
    "Meta Data": {...},
    "Time Series (Daily)": {
      "2026-03-12": {
        "1. open": "2850.00",
        "2. high": "2875.50",
        "3. low": "2840.00",
        "4. close": "2865.25",
        "5. volume": "1500000"
      }
    }
  }

TROUBLESHOOTING

"Rate limited, waiting 5 seconds..."
  → Normal, system will retry automatically

  → Check date range is valid

  → Restart backend
