================================================================================
TROUBLESHOOTING
================================================================================

"Fetching intraday data (attempt 1/3)"
  → Normal, system is fetching data

"Rate limited, waiting 5 seconds..."
  → Normal, system will retry automatically

"No time series data found"
  → Check symbol is valid (IBM not ibm)
  → Check date range is valid
  → Try a different symbol

"API key invalid"
  → Verify key in backend/.env
  → Check for typos
  → Restart backend

"Timeout"
  → Check internet connection
  → Try again
  → Check Alpha Vantage status
