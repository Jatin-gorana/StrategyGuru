WHAT WAS IMPLEMENTED:
================================================================================

1. DSL LEXER (backend/services/dsl_lexer.py)
   - Tokenizes DSL strategy strings
   - Supports all trading indicators and operators
   - Handles syntax validation at token level

2. DSL PARSER (backend/services/dsl_parser.py)
   - Builds Abstract Syntax Tree (AST) from tokens
   - Validates syntax
   - Supports complex expressions with AND, OR, NOT

3. SANDBOX EXECUTOR (backend/services/dsl_sandbox.py)
   - Safely executes AST in isolated environment
   - Pre-calculates all indicators
   - Validates all operations are allowed
   - Blocks malicious code automatically

4. DSL CONVERTER (backend/services/strategy_parser.py)
   - Converts natural language to DSL format
   - Normalizes common trading terms
   - Maintains backward compatibility

5. UPDATED BACKTEST ENGINE (backend/routers/backtest.py)
   - Integrated DSL + Sandbox
   - Automatic fallback to LLM if needed
   - Transparent to users

SUPPORTED INDICATORS:
================================================================================

RSI(period)           - Relative Strength Index
SMA(period)           - Simple Moving Average
EMA(period)           - Exponential Moving Average
MACD()                - MACD indicator
PRICE / CLOSE         - Close price
BOLLINGER(period)     - Bollinger Bands
ATR(period)           - Average True Range
STOCHASTIC(period)    - Stochastic indicator
ADX(period)           - Average Directional Index

================================================================================
SUPPORTED OPERATORS:
================================================================================

Comparison:  <, >, <=, >=, ==, !=
Logical:     AND, OR, NOT

================================================================================
EXAMPLE STRATEGIES:
================================================================================

Simple:
  "Buy when RSI < 30, Sell when RSI > 70"

Intermediate:
  "Buy when RSI < 30 AND price > SMA(200), Sell when RSI > 70"

Advanced:
  "Buy when NOT (RSI > 70) AND SMA(50) > SMA(200), Sell when RSI > 70"
