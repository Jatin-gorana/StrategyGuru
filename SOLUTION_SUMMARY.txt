HOW TO RUN THE PROJECT
================================================================================

FASTEST WAY (Automated):

macOS/Linux: bash run.sh

MANUAL SETUP:
1. cd backend

2. cd frontend (in new terminal)
   npm install
   npm run dev

3. Open http://localhost:3000
