================================================================================
                            HOW IT WORKS
================================================================================

REQUEST FLOW
────────────
1. User submits backtest request
2. Backend computes backtest (~500ms)
3. Backend schedules background task
4. Backend returns response immediately (<1 second)
5. Background task saves to database asynchronously (1-2 seconds)

BACKGROUND TASK
───────────────
1. Create new database session
2. Create strategy record
3. Create backtest record with metrics
4. Bulk insert all trades (1 query)
5. Bulk insert equity curve points (1 query)
6. Single commit for all operations
7. Handle errors gracefully

================================================================================
                         IMPLEMENTATION DETAILS
================================================================================

BULK INSERTS
────────────
Before:
  for trade in trades:
      await create_trade(db, ...)  # 100+ separate queries

After:
  trade_objects = [Trade(...) for t in trades]
  db.add_all(trade_objects)  # 1 bulk query

REDUCED EQUITY CURVE
────────────────────
Before:
  for ec in equity_curve:  # 252 points
      await create_equity_point(db, ...)

After:
  equity_objects = [EquityCurve(...) for ec in equity_curve[::5]]  # 50 points
  db.add_all(equity_objects)

SINGLE COMMIT
─────────────
Before:
  db.add(strategy)
  await db.commit()
  db.add(backtest)
  await db.commit()
  for trade in trades:
      db.add(trade)
      await db.commit()  # Multiple commits

After:
  db.add(strategy)
  db.add(backtest)
  db.add_all(trades)
  db.add_all(equity_points)
  await db.commit()  # Single commit

BACKGROUND EXECUTION
────────────────────
Before:
  background_tasks = None
  await save_to_database()  # Blocks response
  return response

After:
  background_tasks.add_task(save_to_database)  # Non-blocking
  return response  # Immediate
